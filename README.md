# VC Diligence Reports Portfolio

This repository contains a collection of comprehensive Venture Capital due diligence reports executed using the `vc-diligence` expert skill. Each folder represents a deep-dive audit of a specific startup, covering technical architecture, founder alpha, unit economics, market dynamics, and legal risk assessment.

## 📂 Portfolio Structure

- **[cashmere/](./cashmere)**: Infrastructure platform for AI content management and licensing in the publishing industry.

## 🛠️ Methodology

All reports are generated using a **Multi-Wave Recursive Discovery** process:
1. **Wave 1**: Identity, Trajectory & Capital Structure.
2. **Wave 2**: Founder Alpha & Talent Density.
3. **Wave 3**: Technical Architecture & Product Moat.
4. **Wave 4**: Market Dynamics & Competitive Displacement.

Each report includes an **Executive Overview Memo** for partners and a **Technical & Commercial Deep Dive** for exhaustive analysis.
