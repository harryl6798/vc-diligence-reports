# GPTZero: Visual Design Specification (Phase 10.5)

To ensure the visual arguments in our diagrams capture the intimate technical and commercial details discovered in our hyper-depth harvest, we define the following specifications for our primary diagrams.

## 1. Technical Architecture: 7-Layer Ensemble & Origin Telemetry
This diagram must visualize the transition from "Text Product" to "Writing Process."
- **Input Layer**: Dual entry points for "Input Text" and "Origin Telemetry" (Keystroke Dynamics).
- **Hierarchical Engine (The Gated Stack)**: 
    - **L1-L2 (Fast Pass)**: Perplexity and Burstiness engines using distilled models.
    - **L3 (Sentence Precision)**: GPTZeroX sentence-level sliding window.
    - **L4 (Deep Core)**: The multi-billion parameter transformer "Neural Aggregator."
    - **L5 (Bias Mitigation)**: The ESL-tuned Education module.
    - **L6 (Provenance)**: Ground-truth cross-reference against 220M scholarly articles.
    - **L7 (Security)**: The GPTZero Shield (Adversarial defense against homoglyphs/forced errors).
- **Output Verdict**: A trinary classification node (Human / AI / Mixed) leading to the "Writing Report" certificate.
- **Visual Logic**: Use dashed lines for the internal ensemble gating to indicate that not all layers are triggered for every scan (compute efficiency moat).

## 2. Market Dynamics: The Displacement & Data Flywheel
This diagram maps how GPTZero is eating legacy markets while building a new category.
- **Incumbent Node (Slow Giant)**: Turnitin (Opaque/Punitive).
- **GPTZero Node (Disrupter)**: Student-first / Transparency-led (Origin).
- **Displacement Path**: Arrows showing the shift from institutional forensic checking to student-led authorship verification.
- **Segment 1: Academic Integrity**: High-volume wedge (AFT 1.7M member partnership).
- **Segment 2: Enterprise Compliance**: High-margin growth (HR, Legal, Grant Writing).
- **Segment 3: AI Training Integrity**: The "Hidden Moat." Arrows showing 600M human docs flowing into "Verified Ground Truth" for AI labs to prevent "Model Collapse."
- **Flywheel**: A circular arrow showing that more scans -> more ground truth data -> better cleaning for labs -> higher accuracy -> more institutional adoption.

- **X-Axis**: Time (Jan 23 Launch to 2025E).
